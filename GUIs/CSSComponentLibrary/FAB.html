Description

This is a Floating Action Button to be used anywhere on a web page, typically in the bottom left corner.

You can add any icon to the button you would like by changing the material icons 'add' to the name of any icon you find at: Google Material Icons Site

The colors can be changed by changing the colors in the CSS file in the :root{} section.

It is important to remember to link the google stylesheet (if you are going to add an icon), your own CSS file, and your own javaScript file at the top of your HTML file in order for the component to work properly.

CSS

        
:root{
    --lightBlue: #848FA5;
    --darkBlue: #5f6777;
    --pink: #C14953;
    --cream: #E5DCC5;
}
.FAB{
    background-color: var(--pink);
    color: var(--cream);
    border: none;
    border-radius: 50%;
    padding: 20px;
    box-shadow: 0 8px 16px 0 rgba(0,0,0,0.2), 0 6px 20px 0 rgba(0,0,0,0.19);

    display: flex;
    align-items: center;
}
.FAB:hover{
    background-color: var(--darkBlue);
    color: var(--cream);
}
.FAB:active{
    box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2), 0 3px 10px 0 rgba(0, 0, 0, 0.19); 
    background-color: var(--lightBlue);
}
