Description

This is a navigation drawer meant to be fixed to the left of a web page that can be opened and closed.

Any number of links can be added to the drawer by replacing the '#' in the drawer in the HTML with a url of your own or adding an additional a link with the navLink class.

The colors can be changed by changing the colors in the CSS file in the :root{} section.

In addition, you can also change the website title by replacing the 'Your Title Here' text with text of your own.

It is important to remember to link the google stylesheet (for the menu icon), your own CSS file, and your own javaScript file at the top of your HTML file in order for the component to work properly.

JS

            
let isNavOpen = false;

const drawer = document.getElementById("drawer");
const mask = document.getElementById("mask");

function toggleMask() {
    isNavOpen = !isNavOpen;
    drawer.dataset.open = `${isNavOpen}`;
    mask.dataset.open = `${isNavOpen}`;

}

document.getElementById("nav-button").addEventListener("click", () => {
    toggleMask();
});

mask.addEventListener("click", () => {
    toggleMask();
})
