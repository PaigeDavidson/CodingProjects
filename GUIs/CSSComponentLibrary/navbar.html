Description

This is a navigation bar meant to be fixed to the top of a web page.

Up to four links can be added to the bar by replacing the '#' in the links div in the HTML with a url of your own.

You can add any icon to the button you would like by changing the material icons 'add' to the name of any icon you find at: Google Material Icons Site

The colors can be changed by changing the colors in the CSS file in the :root{} section.

In addition, you can also change the website title by replacing the 'Your Title Here' text with text of your own.

It is important to remember to link the Google stylesheet (for the icons), your own CSS file, and your own javaScript file at the top of your HTML file in order for the component to work properly.
