Loading Spinners

Home Navbar Navigation Drawer Button Floating Action Button (FAB) Image Carousel Loading Spinners

Description

These are Loading spinners to be used typically when a user is waiting for something to load.

The colors can be changed by changing the colors in the CSS file in the :root{} section.

The third spinner is interactive and will pause when the user holds the mouse over it which also changes the timing of the bouncing.

It is important to remember to link your own CSS file at the top of your HTML file in order for the component to work properly.

Example 1

HTML


        

        

            
CSS

            
:root{
    --pink: #C14953;
    --cream: #E5DCC5;
    --middle: #c6767c;
}

/* first animation */
.example1{
    display: flex;
    align-items: center;
    gap: 8px;
}

.one {
    width: 50px;
    height: 50px;
    border-radius: 50%;
    background-color: var(--pink);
    animation: example1 .6s infinite;
  }
  .two {
    width: 50px;
    height: 50px;
    border-radius: 50%;
    background-color: var(--pink);
    animation: example1 .6s .2s infinite; 
  }
  .three {
    width: 50px;
    height: 50px;
    border-radius: 50%;
    background-color: var(--pink);
    animation: example1 .6s .4s infinite;
  }
  
  @keyframes example1 {
    0% {
        background-color: var(--pink);
    }
    33% {
        background-color: var(--cream);
    }
    66% {
        background-color: var(--middle);
    }
    100% {
        background-color: var(--pink);
    }
}
